Adding a `git hub` alias

Sun 26 December 2021
by raj
in commandline. Tags: git, github

This git alias will open a web browser and take you to a project's github page when you type git hub in a checked-out project directory.

For OS X:

git config --global alias.hub '!open $(echo $(git config --get remote.origin.url) | perl -pe "s|git\@github\.com:|https://github.com/|")'

For Linux:

git config --global alias.hub '!xdg-open $(echo $(git config --get remote.origin.url) |perl -pe "s|git\@github\.com:|https://github.com/|")'
